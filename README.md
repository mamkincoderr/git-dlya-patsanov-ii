## 😤 Почему ещё одна книга про Git?

Потому что остальные написаны для академиков. А эта — для тех, кто хочет разобраться **сегодня**.

Тимлид сказал *«ребейзни на мейн и открой ПР»* — ты киваешь, но не понимаешь?  
Вводишь `git push` — и что-то идёт не так? **Это твоя книга.**

## 💻 Среда и требования

```
✅ Windows 11          — всё написано и проверено здесь
✅ Git 2.23+           — используем git switch вместо устаревшего git checkout
✅ PowerShell          — не cmd, именно PowerShell
✅ Аккаунт GitHub     — бесплатный, регается за 2 минуты
```

🐧 **Linux** / 🍎 **macOS** — большинство команд те же. Где есть отличия — в тексте есть раскрывающиеся блоки с альтернативными командами.

## 🚀 Как читать

1. Начни с **[Урока 1](chapters/Lesson1.md)** — даже если кажется что знаешь жаргон
2. На каждом уроке есть **практические задания** — делай их, не пропускай
3. Заведи папку `git-practice` — именно в ней и практикуйся
4. Застрял? Открывай [Issue](../../issues) — разберёмся вместе
